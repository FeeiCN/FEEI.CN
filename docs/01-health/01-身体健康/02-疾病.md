# 无病无痛

:::tip[当前疾病]
- **痛风**：已按时吃药控制住，待再次复查。
- **龋齿**：已初步清龋，待进一步清理。
:::

## 核心认知

- **健康是前提，不是附属品。** 身体一旦出问题，工作、收入、计划和心情都会被连带拖垮。
- **疾病来源大致有三部分：基因、环境、行为。** 基因短期难改，真正能持续优化的是环境和行为。
- **很多问题不是突然发生，而是长期积累后显性爆发。** 体重、炎症、代谢、睡眠、压力、饮食和运动习惯都会慢慢推高风险。
- **目标不是活得久，而是尽量长期无病无痛。**

## 我最需要记住的底层规律

- 慢性炎症是很多问题的共同土壤。长期压力、肥胖、睡眠不足、饮食差、缺运动都会推高炎症水平。
- 抗生素、激素、污染环境、深加工饮食、缺少户外和过度久坐，都在持续改变现代人的身体负担。
- 大部分慢性病都和长期生活方式有关，越早干预越便宜。

## 我当前最重要的疾病管理

### 痛风

这是我当前最明确、最需要长期管理的问题。

#### 我对痛风的判断

- 高尿酸不只是“偶尔脚痛”，而是长期代谢和肾脏负担问题。
- 只在急性发作时处理，不足以真正解决问题。
- 真正有效的是 **长期降尿酸 + 饮食控制 + 体重管理 + 水分摄入 + 定期复查**。

#### 当前策略

- 按既往医生方案持续用药，不因为一段时间没发作就擅自停药。
- 重点控制高嘌呤、高果糖、酒精、暴饮暴食、脱水和熬夜。
- 每天喝够水，保持尿液不要长期过浓。
- 通过减重和饮食管理降低复发风险。

#### 必须避免的高风险触发因素

- 海鲜、动物内脏、啤酒、白酒
- 含糖饮料、暴食、连续重口味饮食
- 熬夜、剧烈脱水、发烧后不补水
- 体重持续上升却不处理

#### 急性发作时

- 第一目标是止痛抗炎，第二目标是复盘诱因。
- 是否使用秋水仙碱、NSAIDs 或其他药物，按既往医生方案执行。
- 发作结束后不要就此翻篇，必须回看饮食、作息、体重和水分摄入。

### 口腔

- 龋齿和牙周问题不会自己好，只会拖得更贵、更疼。
- 早晚刷牙、定期洁牙、及时补牙，比拖到根管和拔牙划算太多。

## 我需要长期警惕的共性风险

- **肥胖**：和痛风、脂肪肝、糖尿病、高血压、睡眠质量差高度相关。
- **高血压/高血糖/高血脂**：早期没感觉，不代表没伤害。
- **肝肾负担**：酒精、药物、肥胖、饮食和脱水都会长期拉高风险。
- **压力与睡眠问题**：会放大炎症、情绪和免疫相关问题。

## 其它容易混淆但要记住的事

- 胸痛、明显呼吸困难、单侧无力、突然说不清话、严重意识模糊，优先按急症处理，不自己查笔记慢慢判断。
- 情绪激动、争吵、高原、过度换气后如果出现手麻、头皮发麻、呼吸失衡，先停下来、坐下、慢慢呼吸；如果症状明显或首次出现，直接就医。
- 不用塑料袋或封闭物捂口鼻。

## 保险与保障

下面这些是给未来的自己快速回忆用，属于会变化的信息，每次续保、换工作或政策变化后都要重看。

### 医保

- 基本医疗保险相关规则，见[五险一金](../../03-wealth/04-protection/01-五险一金.md)。

### 商保

- 地区性惠民保：重点看医保外费用的补充报销能力。
- 公司商业保险：重点看门诊、住院、百万医疗、重疾、身故残疾和生育相关保障。

### 记忆点

- 理赔前提往往不是“我觉得自己有病”，而是符合条款定义。
- 购买时如实健康告知，避免未来拒赔。
- 每次换工作都重新核对公司商业保险内容。
