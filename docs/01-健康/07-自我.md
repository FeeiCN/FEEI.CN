# 自我实现

:::tip[当前自我问题]
- 无
:::

**衡量商业成就的标准是金钱，衡量人生成就的标准是幸福**。幸福和钱在早期有直接关系，比如可以带来安全感，包括食物和居所，能避免很多烦恼，也能让自己有更多的自由和机会做觉得有意义的事情。一旦钱超过日常需要后，和幸福关联性就不大。时间上的富裕往往比物质上的富裕更容易让我们活的幸福感。钱是实现幸福的手段，而非目标，不要本末倒置。

**烦恼是常态**。人类祖先越是擅长预见和回避危险，就能活的越久并繁衍更多的后代。时至今日，我们已不用再面对豺狼虎豹，而是工作压力、失业、爱与被爱、冲突、拒绝、疾病、死亡等无数麻烦事，我们经常很难放宽心。

**幸福不是开心，而是一种丰富、满足、知足、充实和有意义的生活体验**。如果只是感觉良好，长期下来反而令人深感不满。用我们真正渴望成为理想自我的方式行动，投入我们深以为然的重要事情，朝我们认为值得的方向前进。为生活注入意义和使命，我们会感觉到自己活的深刻而富有意义。这种感觉不会转瞬即逝，而是强烈感觉到自己在好好生活。

很多烦恼是源自认知问题，当认知升级后，同样的问题会有不一样的看法。

- **乐观**。对所有事情，都有一个合理的解释，能够自洽。都能看到事情的两面性，且在遇到负面事情时，能够看到正向的一面，有塞翁失马。能够从更长周期看待当前事情，有更远大的梦想，眼下的困难就微不足道了。

**不要等待**。幸福不可能三角中，刚工作（想去玩/没钱没时间）、有事业（想去玩/有钱没时间），退休后（玩不动有钱有时间），告诉我们每个阶段有每个阶段的幸福。不要等待解决完各种障碍、完成各种未完成的事情、还完各种债务、存够钱再去生活。你最终会意识到，这些障碍本身就是我的生活。

**把时间和精力投入在对我们最重要和意义的生活领域中，生活就会变得更加丰富充实**。增加想要做的事情，减少不得不做的事情。

**制造对比是让人在健康时感受到幸福的主要方法**。回忆上一次出现问题时的感受、想象自己失去某一部分的感受、感受当下自己身体的状态。

**活在当下**。我们来到这个世界，生不带来，死不带走，唯一留下的是这一生的过程体验。因此珍惜当下，不用懊悔过去，也不必焦虑未来。但这并不意味着是享乐主义，虽然会带来眼前的快乐，但长远来看却是空虚乏味的。也不是另一种勤劳吃苦的极端，过于追求未来的快乐，却承受着现在的痛苦。还有一种无欲无求类型，既不享受现在，也对未来不报有任何期望。而我所追求的幸福是，享受当下，同时我当下所有的行为会让我的未来更幸福。这种追求往往来源于大灾大难后的顿悟，来源于更好视角对生活的认知。


- 自洽
- 积极乐观主义
  - 发生什么不重要，重要的是你怎么看
  - **总能先看到各种事情好的一面**。他的爱好就是打打游戏，这比黄赌毒要好多了。
  - 如果对事情的看法总是坏的一面，比如看待工作时总是看到不公平或者没机会，那么这种底层观念会导致后续的决策和行动出现问题，进入恶性循环，最终影响自己的命运。
- 不追求完美，能看到好的一面，也能接受不好的一面。别人的幸福美满，背后也会有柴米油盐的平淡。巷子里面猫很自由，却没有归宿。围墙里的狗有归宿，却总是要低头。一个男人愿意为你花很多钱，他就不会是个精打细算过日子的人。
- **简化生活**
  - 少看抖音、微博、Twitter
- **活在当下**，人生有太多的不确定，把每一天当成最后一天，每一顿当作最后一顿
  - 花有重开日，人无再少年
  - 落花风雨更伤春，不如怜取眼前人。
  - 欲买桂花同载酒，终不似，少年游。
  - 专注当下可获得内心的宁静和解脱
  - 你是否愿意和90岁的巴菲特交互人生？
- **用视频/文字/语言/图片记录自己的当下**。就像是给自己的纪录片，相信你也不会希望自己看一部索然无味的纪录片，去尝试新奇的冒险、杰出的贡献、各种喜悦瞬间、令人心跳加速的时刻，在未来这些都会是唤起你幸福回忆的时候。
- **不要浪费时间琢磨别人怎么看你，不用活给他人看**。大部分人都活在自己的世界里，并没有人会太关注你。但我们很容易过度在意别人关注自己的看法，因此陷入焦虑。

# 死亡

人们大体会分为四种死亡方式

- 1小时内突然去世；
- 身患绝症，几个月内迅速死亡；
- 器官衰竭；
- 去世前，身体长期处于虚弱状态。

- 人不是到老了才会死，有非常多的死法。
- 避免常见死亡行为：如果我知道我会死在哪里，我就不会去那里。
  - 立即死亡
    - 危险驾驶：开车不系安全带，高速不安全超车，疲劳驾驶，斗气车。自行车/摩托车：不戴头盔，超速行驶，路口未做减速判断。
    - 过马路看手机/分心。
    - 高空抛物，避免走在没有屋檐的路上。
    - 乘坐高危险系数的的交通工具：直升机。
    - 猝死（过量饮酒、加班熬夜）
    - 误食毒物（儿童较多）
    - 窒息（食物/异物堵塞气管）
    - 点击（触摸裸露电线/湿手插电源）
    - 溺水
    - 一氧化碳中毒（燃气漏气）
    - 自杀
  - 不治之症
    - 不安全性行为：艾滋病、梅毒等
    - 长期焦虑/压力环境下：乳腺癌、抑郁症等
    - 长期暴露在污染环境下：甲醛、放射性物质、粉尘等。
- 正确看待死亡
  - 城若破，有死而已，玉可碎，而不可改其白，竹可焚，而不可毁其节，身虽死，名可垂于竹帛也，又何惧哉。
  - 一代人，有一代人的使命。
  - 生命的消亡，但所带来的长远影响将永垂不朽。
