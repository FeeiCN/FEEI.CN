# 注册并订阅AI

0. 使用VPN（https://oixcloud.com/ https://www.fenghost.net//）
1. 通过Google/Apple等方式登陆 Claude Code / Codex
2. 短信验证时使用 https://hero-sms.com
3. 在 https://shop.pockyt.io/ 购买 App Store & iTunes 礼品卡
4. 在 App Store 的个人中心里选择「兑换充值卡或代码」
5. 在 iOS 上打开 Claude Code / Codex App 选择订阅后使用 Apple 账户余额支付
