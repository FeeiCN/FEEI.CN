# X-Tweet-Translator

[X-Tweet-Translator](https://github.com/FeeiCN/X-Tweet-Translator) 是一个用于在 X（Twitter）时间线上自动翻译内容的 Chrome 插件，主打免费使用、无需翻译密钥、自动识别和多语言支持。

![](https://feei.cn/wp-content/uploads/ScreenShot_X-Tweet-Translator-1.webp)

这是一个 AI Coding 实践项目，基于 OpenAI Codex 完成开发与迭代，AI Coding 100%。

## 价值

这个项目的意义不只是做出一个插件，而是验证 AI Coding 在真实项目中的可行性：

- 能否从自然语言需求开始推进功能实现。
- 能否理解浏览器插件的工程结构。
- 能否处理真实页面、DOM、样式和交互问题。
- 能否通过持续迭代修复问题。

## 可复用经验

AI Coding 项目更适合从明确、可观察、可验证的小产品开始。插件类项目边界清晰、反馈直接，很适合作为 AI Coding 的实践入口。
