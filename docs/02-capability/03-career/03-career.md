# 职业与事业

职业发展的核心不是简单升职加薪，而是持续把个人能力、组织影响和商业判断转化为更大的结果。

## 路径

1. **选准行业**：先进入长期有增长空间的赛道。
2. **成为专家**：在关键领域形成专业深度和不可替代性。
3. **拿到结果**：把能力转化为可验证、可评价、可复用的结果。
4. **影响协作**：在没有直接权力时推动他人理解、支持和行动。
5. **管理团队**：通过他人和组织系统放大产出。
6. **创建事业**：从职业能力走向长期事业和企业经营。

## 边界

- **成为专家**关注自己如何变强。
- **拿到结果**关注事情如何做成。
- **影响协作**关注如何让别人支持自己。
- **管理团队**关注如何让一群人持续产出。
