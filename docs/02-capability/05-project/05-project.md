# 作品项目

项目是能力的外化成果。单独学习知识、总结方法还不够，最终要通过具体作品、工具、网站、产品或业务实践，把能力变成真实可用、可展示、可复盘的东西。

## 核心策略

把重要能力尽量沉淀到具体项目里。项目既是练习场，也是作品集；既能验证判断，也能暴露问题，并持续推动能力升级。

## 当前项目

1. [wufeifei.com](./01-wufeifei.com.md)：个人知识管理和人生系统沉淀的试验田。

## 判断标准

- 这个项目是否承载了某项长期能力，而不只是一次性尝试。
- 它是否能持续迭代，并沉淀出可复用的方法、工具或经验。
- 它是否能对外展示自己的判断、审美、执行力和专业能力。
- 它是否和长期目标有关，而不是只追短期新鲜感。
