# 事业有成

事业有成，不只是外在意义上的成功，而是让我觉得人生更有价值和意义的一条路。

它的核心目的，不只是找到一份体面的工作、升职加薪、拿到头衔或创业成功，而是通过长期投入，让自己持续变得更好、能力更强，能够做更值得做的事情，并对未来始终保有清晰的期待。

对我来说，事业有成首先是一种自我塑造：不断提升认知、专业能力、执行力、判断力和影响力，让自己有能力承担更大的责任，解决更重要的问题，创造更真实的价值。

对我来说，事业有成既包括工作，也包括创业。

在工作中，要能够持续成长、成为专家、拿到结果、影响协作、带好团队；如果未来进入创业或更长期的事业阶段，则要把专业能力、产品能力、管理能力和商业判断，转化为可持续的价值创造。

也正因为如此，我希望自己在工作里始终保持开放的心态，愿意学习各种知识，不把自己限制在当前岗位职责范围内。很多当下看似“不直接相关”的知识，长期看都可能在未来串联起来，变成理解业务、理解组织、理解产品、理解用户以及最终创业所需要的关键能力。

## 核心原则

- **先做正确的事**：不是只做紧急的事、容易的事或看起来很努力的事，而是把时间投入到真正重要、长期有价值的事情上。
- **用正确的方式做事**：不是靠蛮力、情绪和短期投机，而是靠系统思考、专业方法、复盘迭代和长期主义。
- **把事情真正做好**：不满足于“参与过”“知道一点”或“差不多”，而是追求可靠、扎实、可复用、能拿结果。
- **持续提升能力**：让自己不断变得更专业、更高效、更有判断力，也更能承担更大范围的责任。
- **保持开放与跨界学习**：不把学习边界锁死在当前职责内，而是主动理解业务、技术、产品、管理和商业，提前为更大的责任和未来创业做准备。

## 能力地图

这个目录记录的，不只是某一项技能，而是通向事业有成的一整套能力系统：

- **[网络空间安全](/security)**：理解攻击、防守、架构、治理与组织建设，在高强度对抗中建立真正有效的安全能力。
- **[软件工程](/software)**：通过工程化、自动化、效率工具和规范化，把想法稳定落地，把复杂事情做成可持续系统。
- **[人工智能](/ai)**：理解 AI 趋势，善用 AI 工具，并把 AI 变成研究、协作、研发和生产力的放大器。
- **[职业与事业](/career)**：从选准行业、成为专家、拿到结果，到影响协作、管理团队、创建事业，形成长期发展的路径。
- **[思维](/thinking)**：提升判断、策略、取舍和博弈能力，在复杂环境中做出更高质量的决策。

此外，真正决定一个人上限的，往往不只是单点技术能力，还包括表达、协作、管理、学习、复盘、商业判断等软性能力。技术让人能做事，软性能力决定能把多大的事做成。
