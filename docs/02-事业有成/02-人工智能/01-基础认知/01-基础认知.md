# 基础认知

基础认知用于沉淀相对稳定的 AI 底层概念，不追逐短期工具热点。

- **[LLM 的技术演进主线](/llm-evolution)**：理解语言模型如何从统计模型、深度学习、Transformer 走向大模型和 Agent。
- **[模型能力边界](/ai-capability-boundary)**：理解 AI 擅长什么、不擅长什么，以及人应该保留哪些判断。
- **[上下文工程](/context-engineering)**：理解如何设计上下文窗口内的信息架构，包括 Few-shot、RAG、KV Cache 和多轮记忆管理。
- **[Agent 架构基础](/agent-architecture)**：理解 Agent 的记忆系统、规划模式、工具调用机制、多智能体协作和可靠性边界。
- **[LLM 对齐与安全](/ai-alignment-safety)**：理解对齐问题的根源、RLHF/CAI/DPO/RLVR 的机制与代价、奉承效应、提示注入、越狱与 Agent 特有安全风险。

这一层回答的问题是：AI 是什么，它为什么有效，它的边界在哪里，以及如何构建在它之上的系统。
