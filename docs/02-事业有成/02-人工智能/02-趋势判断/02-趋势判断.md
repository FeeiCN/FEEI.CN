# AI趋势判断

AI 趋势判断不再放在单篇长文中，而是拆成更稳定的主题：

- **[AI 总体趋势](/ai-trend)**：从传统软件、Workflow、Agent 到新的流量和任务入口。
- **[AI 组织转型](/ai-organization)**：组织如何从围绕“人”转向围绕“事”，以及少数专家和大量 Agent 的协作模式。
- **[AI 与个人发展](/ai-career)**：个人在 AI 时代需要保留和提升的核心能力。
- **[AI 与安全](/ai-security-trend)**：AI 对代码审计、安全行业和 Agent 运行安全的影响。

趋势判断的重点不是追逐工具更新，而是持续回答几个问题：

1. AI 会替代什么？
2. AI 会放大什么？
3. 哪些能力会因为 AI 变得更重要？
4. 组织和个人应该怎样调整工作方式？
