# AI应用实践

AI 应用实践不再按工具品牌堆叠，而是按使用场景组织：

- **[Chat 与 Research](/ai-chat-research)**：如何提问、补充上下文、迭代回答和开展研究。
- **[Agent 实践](/ai-agent-practice)**：Agent 的定义、价值、边界、演进阶段和技术抽象。
- **[Claude](/claude)**：Claude 的模型系列、核心能力、产品形态和 Claude Code 使用方式。
- **[AI Coding](/ai-coding)**：如何使用 Claude Code、Codex 等工具完成真实研发任务。
- **[图片生成](/ai-image-gen)**：图片生成的常见场景、类型和方法。
- **[使用 AI](/ai-use)**：账号订阅、本地模型和 AI 工具使用环境配置。
- **[Skill](/ai-skill)**：如何把经验、流程和判断标准沉淀成可复用资产。
- **[X-Tweet-Translator](/x-tweet-translator)**：AI Coding 的真实项目案例。

实践部分的重点不是记录“某个工具怎么点”，而是沉淀可迁移的方法：

1. 这个场景适合 AI 做什么？
2. 人应该保留哪些判断？
3. 输入、上下文和验证方式应该如何设计？
4. 哪些经验可以沉淀成 Skill、模板或检查清单？
