# 图片生成

图片生成适合用于表达视觉风格、产品形态、场景设定、人物形象、地图、爆炸图、设计蓝图和品牌物料。

常见类型：

- 实物产品。
- 系列品牌物料。
- 城市地标与地图。
- 微缩模型。
- 地标建筑。
- 人物形象。
- 动画角色。
- 产品爆炸图。
- 地形与地标地图。
- 设计蓝图。

## Nano Banana

Nano Banana 2 在高阶知识、文字排版、多图一致性、分辨率和高宽比等方面提升明显。

适合重点探索：

- 真实产品展示。
- 系列品牌一致性。
- 人物形象稳定性。
- 地图与地形可视化。
- 复杂产品结构说明。
- 带文字的设计物料。
