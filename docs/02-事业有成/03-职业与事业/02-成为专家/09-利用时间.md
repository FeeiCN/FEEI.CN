# 利用时间

人在一生中，只有一小部分时间算是真正活着。真正活得久的人，以回忆掌握过去，以理性运用当下，以预见迎接未来——将所有时光合一，拥有最长久的生命。

"随时都能做"的事情根本不存在。行动只在"当下"，只在"此时此刻"——把行动推到"以后"，往往意味着永远不做。

时间是终极生产资料。所有财富，从本质上看都是从时间里挖出来的。钱可以再赚，时间不可逆——保护时间，就是保护最根本的生产能力。

做任何事的真实成本，不只是金钱——还有时间成本和注意力成本。注意力往往是三者中最稀缺的。这个标准同样适用于选择交往的人：此人值得我的时间和注意力吗？

**让时间感更充实的方法**：新体验越多、代谢率越高，时间感越丰富。晨起运动、尝试新事物、积极行动——同样的 24 小时，主动填充的人体感要比被动度过的人长得多。

时间是自己的，没有人有权夺走哪怕一天——除非你允许他们夺走。把心思收回到自己身上，专注修养，不做那些谋求他人认同之事。
