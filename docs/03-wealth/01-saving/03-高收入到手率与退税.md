# 高收入到手率与退税

工作收入不能只看税前数字，真正重要的是最后能到手多少、能留下多少。尤其在高收入阶段，个税、五险一金和各类专项附加扣除，会明显影响最终留存。

## 核心策略

看收入时，不只看税前总数，也要看税后到手、可退税空间和实际留存能力。账面高收入，如果最后留不下来，意义会打折很多。

## 工作收入为什么到手会差很多

- **能拿到较高薪资的人，往往也要面对更高边际税率。** 账面收入增长，不等于到手收入同比例增长。
- **实际到手收入 = 月总收入 - 工资个人所得税 - 五险一金（个人部分）。**
- **高收入群体税前和税后差距通常更大。** 所以在谈薪和评估机会时，也要有税后视角。

## 计税方式记录

- **工资个税计算**：工资 = 本年应税收入合计 x 税率（年度）- 速算扣除（年度）- 往月预扣个税合计。
- **本年应税收入合计** = 本年收入合计 - 本年社保公积金合计（个人）- 本年起征点 - 其它免税合计。
- **税率与速算扣除**：
  - 0-3.6w（3%）
  - 3.6-14.4w（10%，2520）
  - 14.4w-30w（20%，16920）
  - 30w-42w（25%，31920）
  - 42w-66w（30%，52920）
  - 66w-96w（35%，85920）
  - 大于 96w（45%，181920）

## 正规退税和扣除要用好

- **充分利用好正规退税方式。**
- **专项附加扣除**：
  - 子女教育 2k
  - 住房贷款利息 1k
  - 住房租金 1.5k
  - 赡养老人 3k
  - 三岁以下婴儿照顾 2k
  - 大病医疗
  - 继续教育
- **养老金**：每年交 1.2w，退税 5.4k。
- **其它**：高层次人才退税计划等。

## 备注

- 这里保留的是我之前的阶段性理解和记录，具体税率、扣除标准和政策口径要以当年最新规则为准。
- 这篇是对工作收入留存的补充说明，不改变主线判断：高收入窗口期值得珍惜，但更重要的是看最后真正留下了多少。
