# 控制支出

控制支出不是单纯省钱，更不是一味压低生活质量。对我来说，它更像一个顺序明确的消费策略：先不让风险进入系统，再避开会长期吞噬现金流的大坑，最后把钱优先花在真正有价值的地方。

## 核心策略

控制支出，核心不是抠小钱，而是把消费结构整理干净。先守住底线，再管住大额和长期支出，最后把有限的钱花到真正重要的地方。

## 关键指标

- **负债率**：总负债 / 总资产，尤其关注高息负债和消费负债。
- **固定支出占比**：房租、房贷、车、订阅、物业、保险等固定支出 / 月均税后收入。
- **大额支出次数**：年度大额消费是否经过冷静期和复盘。
- **外借资金敞口**：借出去且未收回的钱占净资产比例。

## 这一组内容主要回答三件事

1. **如何不让消费和负债伤到财务系统。** 对应 [零负债，不外借](./01-零负债.md)。
2. **如何避开最容易吞噬现金流的大坑。** 对应 [严控大额或长期支出](./02-严控大额或长期支出.md)。
3. **如何把钱花在更值得的地方。** 对应 [钱优先花在有价值的地方](./03-钱优先花在有价值的地方.md)。

## 我的判断顺序

1. 这笔支出会不会让我背上负担、透支未来，或者把风险转移到自己身上？
2. 它会不会变成长期固定成本，持续吞噬现金流？
3. 如果一定要花，这笔钱能不能真正改善我的生活、能力、健康、关系或未来？

控制支出做到位，重点从来不是“什么都不买”，而是减少低质量支出，把选择权留给自己，把资源留给真正重要的事情。

## 执行规则

- 消费决策先看长期现金流影响，再看当下是否买得起。
- 大额支出默认设置冷静期，并估算买入成本、维护成本、配套成本和退出成本。
- 能变成长期固定成本的支出，要比一次性消费更谨慎。
- 外借资金按照“可能收不回来”来评估，不能伤到自己的应急资金和长期投资计划。
