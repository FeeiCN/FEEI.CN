# 零负债，不借钱，不外借

## 零负债

- **优先使用自己的钱，不借款消费，尤其是高息债务**。
- 注销掉所有信用卡（信用卡的积分累积以及满多少分免年费等都是诱导消费陷阱），将花呗等付款方式优先级调到最低。还清所有消费贷，避免使用各种分期业务，哪怕低利率。
- 负债是一种束缚
- **房贷利息比你想的要高**。
- **不预存消费**。避免向理发店、健身房、教培、饭店会员卡内存钱，非要存也观察该店开店时间、超常规折扣并要严格控制充的数额，避免对方跑路。

## 不借钱给他人

- **所有借钱请求默认回绝。** 当今一个信用正常的人借钱渠道非常多，无论是花呗借呗等网络平台的信用消费产品，还是各个银行的低息无抵押信用贷款，都能随便贷出几十上百万。再不济，他还有他的家人。
- **以下情况绝对不借**
  - 任何形式的消费（买车、买房、买东西等）
  - 任何形式的还贷
  - 没有稳定收入或收入不足以偿还所借款项
  - 过往借过但未及时还款
  - 过往借过未主动提出支付利息
- **以下情况可借**
  - 没钱吃饭
  - 生病住院
  - 借钱生产
- **有办法用钱生钱的人是不会随便借钱出去的，借出去的钱是有机会成本的，能够随意借钱出去往往意味着无知或者无能**

## 不向外面借钱
