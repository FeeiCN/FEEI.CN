# 读书破万卷
> 读书的本质，不是获取信息，而是重构认知与决策能力。

- 认知不足。多数人真正的困境，并非缺钱或缺机会，而是看不清问题本质、做出错误判断、在关键决策上反复犯错。
- 读书三重特性。延迟性，复利性，迁移性。读书的回报不是及时的，而是长期的、潜移默化的。他不会让我立刻变富，但能让我慢慢变得更聪明、更理性、更有判断力。他改变不了我的出身，但能改变我的认知层次，让我拥有应对生活和工作中所有问题的底气。
- 读书的隐性价值。注意力、心性、思考深度。
- 读书是非常高ROI投资。极低的成本，获得别人的答案、错误、经验智慧。
	- 答案：太阳底下没有新鲜事，所有的问题都能在历史中找到答案。


核心策略
- 读什么？
	- 跨界阅读。如果只读某个领域的书，只会导致手里只有锤子，看哪里都像是钉子。
	- 经典。
	- 名人推荐。
	- 高分。
- 怎么读？
	- 带着问题去阅读
	- 用自己的话给他人讲述书中理论
	- 结构化知识沉淀
	- 最小行动清单
