# 人生丰富

健康幸福、事业有成、财务自由，是我很看重的三件事，但我不希望人生只剩下目标、效率和结果。

我也希望自己的人生足够丰富、开阔、有感受力。除了把身体照顾好、把事业做好、把财富系统建立起来，还要持续去体验这个世界，去接触不同的人、不同的地方、不同的思想、不同的审美和不同的生活方式。

对我来说，体验不是单纯地”去过、看过、买过”，而是主动把自己放进真实世界里，去扩大边界、校正判断、积累感受，让人生不至于越活越窄。

判断一件事值不值得去做，有一个简单的标准：临终回望时，它会不会成为那个闪耀着的瞬间？数字和效率不会。

这个部分主要记录我让人生变得更丰富的几种方式：

- **[阅读](/reading)**：通过读书进入他人的经验、思想和时代，提升认知，也提升对世界的理解力。
- **[影视](/film)**：通过电影、剧集、纪录片和历史作品，体会人物、命运、格局、表达与情感。
- **[旅行](/travel)**：通过出发、行走、露营、自驾和远行，走出熟悉环境，去感受真实而广阔的世界。
- **[诗词歌赋](/poetry)**：通过古典文字和各种兴趣爱好，保留审美、好奇心和内在感受力，让生活不只有功能性。

除了这些核心方式，这里也会记录一些围绕生活方式展开的真实实践，比如 [旅行装备](/around-the-world-equipment) 的准备，以及 [房屋设计与装修](/home-design) 中遇到的判断、踩坑与反思。
