# 物品与极简

**每件拥有的物品都会长期占用心智能量。** 即使不使用，也要维护、整理和思考——它构成隐形的生活负担，只是不以账单的形式出现。充满杂物的环境持续消耗感知处理能力，留给真正思考的空间随之减少。

**积累到一定程度，从拥有物品变成被物品拥有。** 每件东西都附带责任：存放它、保养它、为它腾出空间，以及在不再需要时决定如何处置它。物品越多，生活的可移动性和可变动性越低，选择余地随之收窄。

**"便宜货"是幻觉：没有实际用途的东西，价格再低也无价值。** 买一件不会用的东西，等于用负数乘以任何折扣——结果仍然是净损失。"省了多少"只在真正需要的情况下才成立。

## 减少的方式

减少物品不是为了苦行，而是为了让注意力和空间回到真正重要的事情上。

- **只保留真正在用的东西。** 一年内没有使用过的物品，大概率不会再用。
- **先借后买。** 对不确定是否需要的东西，先借用或租用来验证需求，再决定是否购入。
- **单件进，单件出。** 每买入一件新东西，同类中去掉一件旧的。
- **抵抗"以防万一"的冲动。** "以防万一"是大多数无用物品进入生活的理由，而那个万一通常永远不会发生。
