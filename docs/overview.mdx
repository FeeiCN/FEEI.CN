# 创造确定性人生

有一天，看着镜子里的自己——像怀孕的肚子，浮肿的脸部，疲惫的眼神。  
体检报告上的异常指标越来越多，身边人患癌的情况也越来越多。  
自己也要进入上有老下有小的阶段，身边朋友被裁后至今没找到合适的工作。  
还有我那可怜的吉普车，很久没爬山也没过河。  
房贷还了70万，本金却只减少了30万。  
自己能力提升的速度越来越慢了。  
  
忽然的清醒让我意识到：我的内心本能抗拒这样的自己，可我每天的忙碌，似乎并未在解决这些真正重要的问题。  
  
可更多时候，我还是会自我安慰：“还行吧”。  
  
“我其实没那么胖”、“癌症离我还远”、“这岗位不容易被裁”、“贷款总归会还完”、“至少管理能力在提升”。  
  
我努力维持这副“还行”的样子，却心知肚明，真相并没有那么温柔。  
  
还行的身体，意味着没有大病，却远谈不上健康。  
还行的职业，只是能勉强维生，却离梦想遥不可及。  
还行的收入，只是不愁吃穿，却谈不上自由。  
还行的能力，只是平均水准，却离年轻有为相差甚远。  
  
原来，“还行”才是最容易让人停滞不前的陷阱。

  
  
于是，我开始反问自己：  
  
如果继续这样下去，未来会怎样？  
我的身体会不会越来越沉重？  
健康指标会不会雪上加霜？  
会不会哪天，我也是被汰换的那个人？  
房贷什么时候能还完？我何时可以去远行？  
我还有机会年轻有为吗？  
现在的生活，还是我真正想要的吗？  

  
这些问题，我曾害怕面对。  
  
但我明白，不努力前进就会自然后退。  
逃避不解决问题，反而会让问题越来越大。  
  
我必须承认，很多事情正在悄然变糟。  
身体远不如以前了，时常疲惫不堪。  
工作没了当初的快乐，对许多事情失去了热情。  
能力提升放缓，对未来的信心，也变得模糊。  
  
我开始问自己，我到底在追求什么？  
  
不是短期的波动，不是偶然的好运。  
而是能够在复杂变化中依然能找到确定性。  
  
于是，我为人生定下四个长期方向：  

[[健康幸福之路]]、[[事业有成之路]]、[[财务自由之道]]、探索世界
  
它们并非口号，也不是一撮而就的目标。  
更像是四个坐标，帮我在复杂世界中校准方向。  
  
健康幸福，是一切选择的前提。  
事业有成，是能力与价值的体现。  
财务自由，是抵御风险与获得自由的缓冲。  
探索世界，是提醒自己不要只活在效率里。  
  
我不再幻想突如其来的转机，也不再等待最佳时机。  
过去的已是定局，无法再改变，未来还未来。  
我唯一能掌控的，是当下持续的选择。  
现在的每一个选择，正在一点点塑造我的明天。  
未来如何，一定取决于我现在的时间、精力、金钱，投入在何处。  
  
朝着发自内心的方向，把自己全部时间、精力以及资金投入到目标中去，从日常每一个个可执行的步骤开始改变，从一次次决策开始优化，静静等待时机来临，在高胜率情况下勇敢下重注。  
  
  
我不再年轻，却仍有选择的机会。  
我没有大富大贵，却还有追逐的勇气。  
我无法逆转时光，却能决定当下。  
我不能对过去后悔，却能为未来做决定。  
我不想继续麻木，不想看着自己变成一个自己讨厌的人。  
  
从今天起，我愿意正视这些问题，哪怕只是微小的改变。  
只要我朝着目标，每天醒来比昨天更清醒、更坚定，时间会成为我的朋友。  
现在实现这些目标，对于我言，已非“如果”，而是“何时”。  
而时间，成了最可靠的朋友。  
“士别三日，当刮目相看”，将在我的生命中反复上演。  
  
终有一日，在那面镜中，我将遇到更沉稳、更清醒、也更自在的自己。  
  
尽吾志也，而不能至者，可以无悔矣。
